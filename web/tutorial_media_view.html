Media view is a screen that can view the list of media for each album.

Display on the Media View

The Album is a list of media types from a particular folder in the system.

slide the screen to the right and panel view will open. you can navigate easily.

Cross-browser compatibility

If you long press the media, it will switch to multi-select mode. You can work various operations - copy, move, or share etc.

screen menu

  • play : play the viewer for the media.
  • view type : The format of the display is changed in the form of simple, detail and preview type.
  • merge(grouping) : You can bind your media by group.
  • sort : a list of the albums is sorted by name, date or path. In all the category it is applied overall.
  • setting : You can set a theme, menu-type, lock type.
  • finish : this App is exited.

album menu

  • multi-select : swich selected mode by long press one of medias. You can add your selection by clicking the different media.
  • paste more : In the selected mode, tap a album in the panel and you can move or copy the selected meidas.
  • share

    You can share your medias with SNS or other applications.

  • rename

    You can rename selected albums.

  • delete

    You cna delete selected albums.

  • select all

    You can select or unselect all.

  • lock

    hide and protect your medias.
